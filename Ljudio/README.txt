-=ABOUT LJUDIOPLAYER=-

PROJEKTUPPGIFT “LJUDIO”

Du ska leverera en interaktiv webbapplikation, där användarens
primära mål ska vara att kunna söka efter musik-titlar och artister 
och sedan kunna lyssna på en låt.


-=HOW TO INSTALL=-

npm install

npm install axios  

npm install react-player   

npm install react-router-dom

npm install @mui/icons-material

npm install @mui/material @emotion/react @emotion/styled

-=HOW TO RUN=-

npm run dev
